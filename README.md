# 🌐 Language Translator

## 🚀 Overview
A modern and responsive Language Translator web application built using React. It allows users to translate text between multiple languages in real-time using an external API.

## ✨ Features
- 🌍 Supports multiple languages
- ⚡ Real-time translation
- 🔁 Language swap functionality
- 🧠 Built using React Hooks (useState, useEffect)
- 📱 Responsive and clean UI

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

## 🔌 API Used
- MyMemory Translation API  
  https://mymemory.translated.net/

# 🌐 Language Translator

## 📌 Description
Ye ek web-based language translator app hai jo ek language se doosri language me text convert karta hai.

## 🚀 Features
- Multiple language support
- Real-time translation
- Simple and clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- React
- API (Google Translate API / LibreTranslate)

## ⚙️ How to Run
1. Clone the repo
2. Open index.html in browser
